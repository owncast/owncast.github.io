Owncast is a self-hosted live video and web chat server for use with existing popular broadcasting software.

Get started

Self hosted

Why self-hosted live video is great. If you're currently streaming it's likely you can point your existing video software at your Owncast server and be ready to go. Try it!

Chat Stuff

The power the web chat server. Owncast works with different storage providers to optionally scale your videos to many viewers using only low-powered hardware.

Works with Software

Use with existing popular broadcasting software. The Owncast community is vibrant and always ready to help. Visit us for support or if you just want to hang out.

Storage

Owncast is licensed under the MIT license, so you can use it freely to build anything you like. Even better, you can extend the functionality of your Owncast server with 3rd party APIs.
