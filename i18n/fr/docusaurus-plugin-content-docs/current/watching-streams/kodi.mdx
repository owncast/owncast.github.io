    <ul>
      <li>Fonctionne avec n'importe quelle installation de Kodi.</li>
      <li>Accès direct à des serveurs Owncast spécifiques.</li>
      <li>Aucun plugin supplémentaire requis.</li>
    </ul>

    <details>
      <summary>Afficher les instructions étape par étape</summary>

      1. Dans Kodi, allez à **Vidéos** > **Fichiers**.
      2. Sélectionnez **Ajouter des vidéos**.
      3. Choisissez **Parcourir** puis **Ajouter une localisation réseau**.
      4. Entrez l'URL de votre serveur Owncast (par exemple, `https://watch.owncast.online`).
      5. Enregistrez l'emplacement et accédez-y pour lire le flux.
