    <ul>
      <li>Compatible avec tout flux Owncast dans le navigateur Chrome.</li>
      <li>Processus de diffusion simple en quelques clics.</li>
      <li>Aucune application supplémentaire requise sur le Chromecast.</li>
    </ul>

    <details>
      <summary>Afficher les instructions étape par étape</summary>

      1. Sur votre ordinateur, ouvrez **Chrome**.
      2. Visitez le serveur Owncast que vous souhaitez regarder.
      3. En haut à droite, cliquez sur le bouton **Plus** puis sur **Diffuser**.
      4. Choisissez le dispositif Chromecast sur lequel vous souhaitez regarder le contenu.
      5. Si vous utilisez déjà le Chromecast, votre contenu remplacera ce qui est affiché sur votre télévision.

    </details>

    Pour plus de détails, visitez la [documentation officielle de Chromecast](https://support.google.com/chromecast/answer/3228332).
