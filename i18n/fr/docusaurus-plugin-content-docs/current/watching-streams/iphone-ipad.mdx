    <ul>
      <li>Aucune installation d'application requise.</li>
      <li>Fonctionne avec n'importe quel serveur Owncast.</li>
      <li>Contrôles AirPlay intégrés dans le lecteur vidéo.</li>
      <li>Support de l'image dans l'image sur les appareils compatibles.</li>
    </ul>

    <details>
      <summary>Afficher des instructions étape par étape</summary>

      1. Ouvrez **Safari** sur votre iPhone ou iPad.
      2. Accédez à l'URL de votre serveur Owncast (par exemple, `https://watch.owncast.online`).
      3. Appuyez sur le bouton lecture pour commencer le flux.
      4. Utilisez le bouton AirPlay dans les contrôles vidéo pour caster vers d'autres appareils.
      5. Appuyez sur le bouton plein écran pour une vision immersive.

    </details>

  </DeviceOptionCard>

  <DeviceOptionCard id="vlc-ios" title="VLC pour iOS" difficulty="moyen" reliability="élevé" screenshot="/images/devices/iphone-ipad/vlc-network.png" screenshotAlt="Interface de flux réseau VLC pour iOS" summary="Utilisez VLC pour lire les flux Owncast avec des contrôles de lecture avancés et des options.">








    <ul>
      <li>Lecteur multimédia avancé avec support de format étendu.</li>
      <li>Support des flux réseau pour l'entrée d'URL directe.</li>
      <li>Fonctionnalités de lecture hors ligne.</li>
      <li>Support AirPlay intégré.</li>
    </ul>

    <details>
      <summary>Afficher des instructions étape par étape</summary>

      1. Installez **VLC pour iOS** depuis l'App Store.
      2. Ouvrez VLC et appuyez sur l'onglet **Réseau**.
      3. Entrez l'URL de votre serveur Owncast (par exemple, `https://watch.owncast.online`).
      4. Appuyez sur le flux pour commencer la lecture.
      5. Utilisez le bouton AirPlay pour caster vers d'autres appareils, si désiré.

    </details>

    <StoreButton url="https://apps.apple.com/app/vlc-for-mobile/id650377962" type="ios" />

  </DeviceOptionCard>

  <DeviceOptionCard id="airplay-casting" title="AirPlay vers TV" difficulty="facile" reliability="élevé" screenshot="/images/devices/iphone-ipad/airplay-tv.png" screenshotAlt="Écran de sélection des appareils AirPlay" summary="Diffusez depuis votre iPhone ou iPad vers Apple TV, télévisions compatibles AirPlay ou autres récepteurs AirPlay.">








    <ul>
      <li>Fonctionne à la fois depuis l'application Owncasts et Safari.</li>
      <li>Diffusion vidéo de haute qualité sur de grands écrans.</li>
      <li>Contrôlez la lecture depuis votre appareil mobile.</li>
      <li>Poursuivez l'utilisation de votre téléphone pendant la diffusion.</li>
    </ul>

    <details>
      <summary>Afficher des instructions étape par étape</summary>

      1. Commencez à lire un flux Owncast sur votre iPhone ou iPad (en utilisant soit l'application Owncasts soit Safari).
      2. Recherchez le bouton **AirPlay** dans les contrôles vidéo ou l'interface de l'application.
      3. Appuyez sur le bouton AirPlay et sélectionnez votre appareil cible (Apple TV, smart TV, etc.).
      4. Le flux commencera à jouer sur l'appareil sélectionné.
      5. Utilisez votre iPhone/iPad comme télécommande.

    </details>

    Pour plus de détails, consultez [la documentation AirPlay d'Apple](https://support.apple.com/en-us/HT204289).

  </DeviceOptionCard>

  <DeviceOptionCard id="control-center-airplay" title="AirPlay du Centre de Contrôle" difficulty="facile" reliability="élevé" screenshot="/images/devices/iphone-ipad/control-center-airplay.png" screenshotAlt="Centre de contrôle iOS avec le bouton AirPlay" summary="Utilisez le Centre de Contrôle pour diffuser rapidement tout flux Owncast vers des appareils compatibles.">








    <ul>
      <li>Accès AirPlay système depuis le Centre de Contrôle.</li>
      <li>Fonctionne avec n'importe quelle application ou navigateur jouant des flux Owncast.</li>
      <li>Changement rapide d'appareil.</li>
    </ul>

    <details>
      <summary>Afficher des instructions étape par étape</summary>

      1. Commencez à jouer un flux Owncast dans n'importe quelle application ou Safari.
      2. Ouvrez **Centre de Contrôle** (faites glisser vers le bas depuis le coin supérieur droit sur les iPhones récents, faites glisser vers le haut depuis le bas sur les modèles plus anciens).
      3. Appuyez sur l'icône **AirPlay** (rectangle avec triangle).
      4. Sélectionnez votre appareil compatible AirPlay dans la liste.
      5. L'audio et la vidéo seront diffusés vers l'appareil sélectionné.

    </details>

  </DeviceOptionCard>
