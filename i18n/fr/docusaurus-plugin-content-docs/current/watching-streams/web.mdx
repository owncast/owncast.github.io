## Lecteur Web

<img src="/docs/img/owncast-web-interface.png" alt="L'interface web d'Owncast affichant le lecteur vidéo, le titre du flux, les boutons de suivi et de notification, et le chat en direct" width="80%" />

Le lecteur vidéo web d'Owncast dispose de plusieurs raccourcis clavier que vous pouvez utiliser.

| Action                  | Raccourci        |
| ----------------------- | ---------------- |
| Lecture/Pause           | *Barre d'espace* |
| Augmenter le volume     | *0*              |
| Diminuer le volume      | *9*              |
| Muet                    | *m*              |
| Basculer en plein écran | *f*              |
| Basculer le chat        | *c*              |
