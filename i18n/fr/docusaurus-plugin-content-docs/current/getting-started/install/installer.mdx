En utilisant l'installateur, vous pouvez être opérationnel en environ une minute. Cependant, si vous préférez [télécharger une version manuellement](/docs/getting-started/install/manual), utiliser une [installation en un clic](/docs/getting-started/install/providers) chez un fournisseur d'hébergement pris en charge, ou d'autres méthodes, visitez la [page d'installation](/docs/getting-started/install/) pour trouver la méthode d'installation qui correspond le mieux à vos besoins.

### Installateur

Collez ce qui suit dans votre shell et l'installateur téléchargera la version la plus récente d'Owncast pour votre plateforme.
Il téléchargera également une copie de ffmpeg si vous n'en avez pas actuellement installé.

```bash
curl -s https://owncast.online/install.sh | bash
```

**Remarque** : Il n'est pas conseillé d'exécuter cela en tant que root, car vous téléchargez et exécutez un script depuis Internet. Il est également recommandé d'inspecter le contenu de tout script distant avant de l'exécuter.
