Il existe plusieurs façons d'installer Owncast. L'installateur rapide ci-dessous est le plus simple et fonctionne sur la plupart des serveurs Linux et macOS. Vous pouvez également installer [manuellement](#manual-installation) à partir d'un téléchargement de version, exécuter l'[image du conteneur](/docs/getting-started/install/container), ou utiliser une [installation en un clic auprès d'un fournisseur d'hébergement](/docs/getting-started/install/providers).
