Le chat de votre flux peut être personnalisé avec vos propres émotes. Vous pouvez en ajouter autant que vous le souhaitez et elles seront disponibles pour être utilisées dans le chat.

## Dans l'Admin

En visitant la page des émotes personnalisées dans l'admin située à `/admin/chat/emojis`, vous pouvez ajouter et supprimer des images disponibles pour les participants au chat.

<img src="/docs/img/admin-custom-emoji.png" alt="La page des émotes personnalisées dans l'admin, avec un bouton Télécharger une émote personnalisée au-dessus d'une grille des émotes actuellement installées" width="60%" />

## Gestion manuelle

Vous pouvez gérer vos images d'émotes personnalisées dans le répertoire `data/emoji` de votre système de fichiers. Voici ce que vous pouvez faire :

- Ajouter plusieurs fichiers d'images d'émotes statiques et/ou animées à la fois
- Supprimer un ensemble entier d'émotes
- Renommer tous les fichiers d'émotes
- Sauvegarder vos émotes personnalisées en les copiant dans un autre dossier

## Images d'émotes groupées

Les images groupées avec Owncast par défaut sont librement sous licence par différents auteurs. Voir le [répertoire des émotes](https://github.com/owncast/owncast/tree/develop/static/img/emoji) pour les licences respectives associées à chaque collection.
