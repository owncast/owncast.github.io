### Mise en forme du texte

Le chat web prend en charge un formatage de base en utilisant [Markdown](https://www.markdownguide.org/basic-syntax/):

Italique: `*votre texte*`

Gras: `**votre texte**`

Barré: `~~votre texte~~`

Blocs de code: <code>\`votre texte\`</code>

Lorsque vous envoyez un message, Owncast rend le formatage en ligne :
