## Vue d'ensemble

Owncast comprend une interface web pour votre vidéo avec un chat intégré qui est disponible une fois que vous avez démarré le serveur. Elle montre les états en ligne/hors ligne, le nombre de spectateurs, la durée du flux, la description de votre instance, des images, des liens et plus encore. Vous pouvez commencer à l'utiliser sans apporter de modifications, mais vous voudrez probablement mettre à jour le contenu affiché sur votre page en visitant votre page d'administration du serveur.

<img src="/docs/img/owncast-web-interface.png" alt="The Owncast web interface showing the video player, stream title, follow and notify buttons, and live chat" width="80%" />

De plus, l'interface web a été spécialement conçue pour être personnalisable par quiconque se sent à l'aise pour modifier les couleurs et les styles. Aucun environnement de développement n'est nécessaire, il suffit d'ouvrir l'admin et de commencer à ajuster.

Si vous souhaitez intégrer Owncast dans votre site web existant, consultez notre [documentation sur l'intégration de Owncast](/docs/embed/).

Voici quelques éléments que vous voudrez probablement personnaliser pour mettre à jour le contenu affiché sur votre page. Vous les trouverez tous sur la page des paramètres **Généraux** dans l'admin.

<img src="/docs/img/admin-website-general.png" alt="The General settings page in the admin, with fields for the instance name, description, logo, tags, social links, and custom page content" width="65%" />

### Nom et description

En définissant votre nom, votre description et votre logo, vous pouvez rapidement mettre à jour le contenu du site web pour refléter votre flux.

### Tags

En définissant des tags, vous montrez aux spectateurs potentiels les catégories de contenu que vous diffusez généralement.

### Liens sociaux externes

Vous pouvez ajouter des liens vers vos profils sur d'autres sites en les ajoutant dans l'administration.

### Contenu de la page web

Le corps du contenu de votre page peut être personnalisé dans votre administration. Utilisez la [syntaxe Markdown standard](https://www.markdownguide.org/basic-syntax/) pour ajouter des liens, des images, et plus encore.
