---

Si vous avez du Javascript à exécuter lorsque votre page web Owncast se charge, vous pouvez l'ajouter à l'éditeur de Javascript dans la page des paramètres **Général** dans l'administration.

<img src="/docs/img/admin-custom-javascript.png" alt="L'onglet Script personnalisé de la page des paramètres généraux, montrant l'éditeur de Javascript" width="80%" />

Voici quelques exemples de raisons pour lesquelles vous pourriez en avoir besoin :

- Ajouter des analyses respectueuses de la vie privée.
- Initialiser un processeur de paiement que vous avez intégré sur votre page.
- Utilisez Javascript pour manipuler l'interface.

:::warning

Vérifiez votre Javascript. Toute syntaxe incorrecte ou erreurs que vous insérez dans la page peuvent créer des erreurs et empêcher le chargement de la page ou le fonctionnement des fonctionnalités.
:::
