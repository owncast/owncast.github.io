Vous devez être un utilisateur payant de [Restream](http://restream.io) pour pointer vers votre instance Owncast en tant que "canal" de destination.

1. Connectez-vous et allez à l'écran "Ajouter un canal".
   ![Écran Ajouter un canal](/docs/img/restream1.png)

2. Sélectionnez "RTMP personnalisé"
   ![Sélectionnez RTMP personnalisé](/docs/img/restream2.png)

3. Ajoutez les informations de votre serveur au format `rtmp://monserveur/live` pour l'URL RTMP et votre Clé de diffusion.
   ![Informations sur le serveur](/docs/img/restream3.png)
