[Jitsi](https://jitsi.org) est à la fois un fournisseur de vidéoconférence et une suite logicielle. Il est open source et peut être auto-hébergé. Il est également disponible en tant que service sur [meet.jit.si](https://meet.jit.si).

1. Visitez votre page de réunion Jitsi.
2. Cliquez sur les trois points dans le coin inférieur droit.
3. Sélectionnez "Démarrer le live stream"

![Image](/docs/img/jitsi-1.png)

4. Dans la fenêtre modale "Démarrer un écran en direct", mettez votre RTMP Owncast complet dans le champ "Clé de live stream". Cela serait dans le format rtmp://your-owncast-server/live/yourkey.

![Image](/docs/img/jitsi-2.png)

5. Notez que l'erreur "La clé du live stream peut être incorrecte" s'affichera. C'est attendu et peut être ignoré.
