## Matériel

Il est important de savoir comment votre CPU, en particulier, gère les tâches de traitement vidéo lors du dépannage. Owncast vous donne un aperçu général des métriques matérielles clés pour vous aider avec cela.

## Prometheus

Pour les personnes qui préfèrent utiliser des solutions de surveillance externes, Owncast prend en charge l'utilisation de [Prometheus](https://prometheus.io/) pour collecter un ensemble de métriques.

Vous pouvez diriger la configuration de Prometheus vers votre instance Owncast avec le point de terminaison de `/api/admin/prometheus`, en utilisant l'authentification de base et les données de connexion administratives.

Par exemple :

```yaml
- job_name: owncast
  scrape_interval: 1m
  metrics_path: /api/admin/prometheus
  scheme: https
  basic_auth:
    username: admin
    password: my_admin_password
  static_configs:
    - targets: ["my.owncast.server"]
```
