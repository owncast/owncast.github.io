Visitez la page d'utilisation du matériel dans l'administration pour voir comment les ressources de votre serveur fonctionnent.

<img src="/docs/img/admin-hardware-info.png" alt="La page d'information sur le matériel dans l'administration, affichant des jauges d'utilisation de CPU, de mémoire et de disque avec un graphique d'utilisation dans le temps" width="80%" />

Si votre CPU est saturé sur votre serveur Owncast, les spectateurs verront des mises en mémoire tampon, car le serveur ne peut plus suivre. Si c'est ce que vous ressentez, la première étape est de [résoudre l'utilisation excessive du matériel](/docs/troubleshoot/hardware-usage).
