1. Assurez-vous d'avoir [le passtrough vidéo](/docs/video/#video-passthrough) **désactivé**. Je sais que vous voulez le garder activé, mais si vous avez des problèmes, **DÉSACTIVEZ-LE**. Si cela ne résout pas le problème, vous pourrez le réactiver plus tard lorsque vous aurez résolu le problème.
2. Regardez vos journaux Owncast ainsi que `transcoder.log` pour voir s'il y a des erreurs qui pourraient être utiles.
3. Pour résoudre le problème, essayez une autre source et/ou un autre lecteur afin que, lorsque vous demanderez de l'aide, vous sachiez que cela fonctionne pour X mais ne fonctionne pas pour Y. Votre flux est disponible à `/hls/stream.m3u8`, essayez donc de le mettre dans VLC, Quicktime, etc.
4. Si vous utilisez un service de re-diffusion tiers ou un service de streaming basé sur le web, essayez de diffuser sans cela pour voir si ça fonctionne. Encore une fois, cela aide lorsque vous demandez de l'aide pour nous faire savoir où se trouve le problème.
