Vous aurez peut-être plus de chance d'exécuter Owncast sur un serveur qui pourrait être plus centralement situé géographiquement ou qui a une connexion au monde plus rapide. Si votre serveur est physiquement éloigné de vous (pour diffuser) et de vos spectateurs (pour consommer la vidéo), vous n'êtes peut-être pas dans la configuration la plus optimale.
