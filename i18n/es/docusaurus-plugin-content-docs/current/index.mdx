## Instalación

Si buscas instalar Owncast por primera vez, pasa por nuestro Asistente de Inicio Rápido para comenzar a funcionar. Te guiará a través de las opciones de instalación disponibles para tus casos de uso. [Asistente de Inicio Rápido](/quickstart)

## Tu primera transmisión

Una vez que tengas Owncast en funcionamiento, querrás prepararte para tu primera transmisión. Consulta la guía de [Configura tu clave de transmisión y tu primera transmisión](/docs/getting-started/configure-first-stream) para aprender a configurar tu software de transmisión y comenzar a transmitir.

## Configuración

Owncast ofrece una variedad de opciones de configuración para personalizar tu experiencia de transmisión. Visita la página de Configuración para explorar los diferentes ajustes disponibles, incluyendo calidad de video, personalización de página, características sociales y más. [Configuración](/docs/configuration)

## Solución de problemas

¿Tienes preguntas o has encontrado problemas? El Asistente de Solución de Problemas proporciona soluciones a problemas comunes y respuestas a preguntas frecuentes. [Asistente de solución de problemas](/troubleshoot)

## Construyendo sobre Owncast

¿Buscas construir bots, superposiciones, integraciones, herramientas u otra funcionalidad personalizada? Consulta la [Referencia de la API](/api/release) para obtener detalles sobre cómo interactuar con Owncast de forma programática.

## ¿Quieres contribuir a esta documentación?

¡Owncast es un proyecto de código abierto y las contribuciones a la documentación son bienvenidas! If you find any errors, have suggestions for improvements, or want to add new content, please do help us make the documentation better for everyone! Puedes contribuir enviando una solicitud de extracción en el [repositorio de documentación de Owncast](https://github.com/owncast/owncast.github.io). Recibirás reconocimiento en la página que ayudaste a mejorar.
