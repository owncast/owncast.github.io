---
title: Agregar Javascript personalizado
description: Ejecuta Javascript personalizado en tu página web de Owncast.
sidebar_label: Agregar Javascript personalizado
---

Si tienes algún Javascript que necesitas ejecutar cuando se carga tu página web de Owncast, puedes añadirlo al editor de Javascript en la página de configuración **General** en el administrador.

<img src="/docs/img/admin-custom-javascript.png" alt="La pestaña de scripting personalizado de la página de configuración General, mostrando el editor de Javascript" width="80%" />

Algunos ejemplos de razones por las cuales podrías necesitar esto:

- Agregar análisis que respeten la privacidad.
- Inicializar un procesador de pagos que hayas incrustado en tu página.
- Usa Javascript para manipular la interfaz.

:::warning

Verifica tu Javascript. Cualquier sintaxis incorrecta o errores que insertes en la página pueden crear errores y detener la carga de la página o el funcionamiento.
:::
