## Resumen

Owncast incluye una interfaz web para tu video con chat integrado que está disponible una vez que inicias el servidor. Muestra los estados en línea/fuera de línea, el recuento de espectadores, la duración de la transmisión, la descripción de tu instancia, imágenes, enlaces y más. Puedes comenzar a usarlo sin hacer cambios, pero probablemente querrás actualizar el contenido que se muestra en tu página visitando la página de administración de tu servidor.

<img src="/docs/img/owncast-web-interface.png" alt="The Owncast web interface showing the video player, stream title, follow and notify buttons, and live chat" width="80%" />

Además, la interfaz web fue construida específicamente para ser personalizable por cualquier persona que se sienta cómoda ajustando colores y estilos. No se necesita un entorno de desarrollo, solo abre el panel de administración y comienza a ajustarlo.

Si deseas incrustar Owncast en tu sitio web existente, consulta nuestra [documentación sobre la incrustación de Owncast](/docs/embed/).

A continuación se presentan algunos elementos que probablemente desees personalizar para actualizar el contenido que se muestra en tu página. Los encontrarás todos en la página de configuración **General** en el panel de administración.

<img src="/docs/img/admin-website-general.png" alt="The General settings page in the admin, with fields for the instance name, description, logo, tags, social links, and custom page content" width="65%" />

### Nombre y descripción

Al establecer tu nombre, descripción y logotipo, puedes actualizar rápidamente el contenido del sitio web para reflejar tu transmisión.

### Etiquetas

Al establecer etiquetas, estás mostrando a los espectadores potenciales qué categorías de contenido sueles transmitir.

### Enlaces sociales externos

Puedes agregar enlaces a tus perfiles en otros sitios añadiéndolos en el panel de administración.

### Contenido de la página web

El contenido principal de tu página se puede personalizar en tu panel de administración. Utiliza la [sintaxis estándar de Markdown](https://www.markdownguide.org/basic-syntax/) para agregar enlaces, imágenes y más.
