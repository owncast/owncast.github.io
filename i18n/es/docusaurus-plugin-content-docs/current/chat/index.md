### Formato de Texto

El chat web soporta algunos formatos básicos usando [Markdown](https://www.markdownguide.org/basic-syntax/):

Cursiva: `*tu texto*`

Negrita: `**tu texto**`

Tachado: `~~tu texto~~`

Bloques de código: <code>\`tu texto\`</code>

Cuando envías un mensaje, Owncast renderiza el formato en línea:
