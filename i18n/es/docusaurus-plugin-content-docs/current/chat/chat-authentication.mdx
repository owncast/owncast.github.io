No hay requisito para autenticarte al usar el chat de Owncast. Sin embargo, algunos prefieren
autenticarse para verificar su identidad ante otros, y para continuar usando la misma identidad de chat a través de múltiples dispositivos y navegadores. Esto es especialmente útil para aquellos con privilegios de [moderador](/docs/chat/).

Puedes acceder a las opciones de autenticación a través del menú desplegable de chat en la parte superior derecha de la página.

## Autenticación del Fediverso (FediAuth)

Usando tu cuenta de Fediverso, puedes recibir un código de un solo uso para verificar tu identidad y autenticarte con el chat de Owncast. El soporte de Fediverso debe estar habilitado en el servidor de Owncast para que esta función esté disponible.

Esto se hace enviando un mensaje directo a tu cuenta. Si no recibes este mensaje, asegúrate de que puedes aceptar
mensajes directos.

Estos códigos expiran, por lo que necesitarás solicitar uno nuevo si es necesario.
  </TabItem>

  <TabItem value="IndieAuth" label="IndieAuth">
    ## IndieAuth

IndieAuth es un protocolo de autenticación descentralizado de estándar abierto que permite a los servicios verificar la identidad de un usuario representado por una URL. Esto significa que cualquiera que tenga una URL existente que soporte IndieAuth puede usarla para autenticarse en el chat de Owncast.

Visita [IndieAuth.net](https://indieauth.net/) para aprender más.

### Owncast es un servidor IndieAuth

Si administras un servidor Owncast, puedes usarlo para autenticarte en otras instancias de Owncast utilizando la URL de tu servidor.
