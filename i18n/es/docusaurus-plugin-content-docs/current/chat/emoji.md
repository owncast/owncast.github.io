El chat de tu transmisión se puede personalizar con tus propios emojis personalizados. Puedes agregar tantos como desees y estarán disponibles para usar en el chat.

## En el Admin

Al visitar la página de emojis personalizados en el admin ubicada en `/admin/chat/emojis`, puedes agregar y eliminar imágenes que están disponibles para los participantes del chat.

<img src="/docs/img/admin-custom-emoji.png" alt="La página de emojis personalizados en el admin, con un botón de 'Subir nuevo emoji' sobre una cuadrícula de los emojis actualmente instalados" width="60%" />

## Gestión manual

Puedes gestionar tus imágenes de emoji personalizadas en el directorio `data/emoji` de tu sistema de archivos. Esto es lo que puedes hacer:

- Agregar múltiples archivos de imágenes de emoji estáticos y/o animados a la vez
- Eliminar un conjunto completo de emojis
- Renombrar todos los archivos de emojis
- Haz una copia de seguridad de tus emojis personalizados copiándolos a otra carpeta

## Imágenes de emojis empaquetados

Las imágenes empaquetadas con Owncast vienen con una licencia gratuita de diferentes autores. Consulta el [directorio de emojis](https://github.com/owncast/owncast/tree/develop/static/img/emoji) para las respectivas licencias asociadas a cada colección.
