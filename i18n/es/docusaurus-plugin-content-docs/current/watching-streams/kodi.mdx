    <ul>
      <li>Funciona con cualquier instalación de Kodi.</li>
      <li>Acceso directo a servidores específicos de Owncast.</li>
      <li>No se requieren plugins adicionales.</li>
    </ul>

    <details>
      <summary>Mostrar instrucciones paso a paso</summary>

      1. En Kodi, ve a **Videos** > **Archivos**.
      2. Selecciona **Añadir Videos**.
      3. Elige **Navegar** y luego **Añadir Ubicación de Red**.
      4. Ingresa la URL de tu servidor Owncast (por ejemplo, `https://watch.owncast.online`).
      5. Guarda la ubicación y accede a ella para reproducir el flujo.

    </details>
