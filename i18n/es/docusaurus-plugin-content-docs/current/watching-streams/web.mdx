## Reproductor Web

<img src="/docs/img/owncast-web-interface.png" alt="La interfaz web de Owncast mostrando el reproductor de video, el título de la transmisión, los botones de seguir y notificar, y el chat en vivo" width="80%" />

El reproductor de video web de Owncast tiene una serie de atajos de teclado que puedes usar.

| Acción                     | Atajo               |
| -------------------------- | ------------------- |
| Reproducir/Pausar          | *Barra espaciadora* |
| Subir volumen              | *0*                 |
| Bajar volumen              | *9*                 |
| Silenciar                  | *m*                 |
| Alternar pantalla completa | *f*                 |
| Alternar chat              | *c*                 |
