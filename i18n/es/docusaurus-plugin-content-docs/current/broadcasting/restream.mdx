Debes ser un usuario de pago de [Restream](http://restream.io) para señalar tu instancia de Owncast como un "canal" de destino.

1. Inicia sesión y ve a la pantalla de "Agregar Canal".
   ![Pantalla de Agregar Canal](/docs/img/restream1.png)

2. Selecciona "RTMP Personalizado"
   ![Seleccionar RTMP Personalizado](/docs/img/restream2.png)

3. Agrega la información de tu servidor en el formato `rtmp://myserver/live` para la URL RTMP y tu clave de transmisión.
   ![Información del Servidor](/docs/img/restream3.png)
