Al usar el instalador, puedes empezar a funcionar en aproximadamente un minuto. Sin embargo, si prefieres [descargar una versión manualmente](/docs/getting-started/install/manual), usar una [instalación con un clic](/docs/getting-started/install/providers) en un proveedor de alojamiento compatible, o otros métodos, visita la [página de instalación](/docs/getting-started/install/) para encontrar el método de instalación que mejor se adapte a tus necesidades.

### Instalador

Pega lo siguiente en tu terminal y el instalador descargará la versión más reciente de Owncast para tu plataforma.
También descargará una copia de ffmpeg si no tienes uno instalado.

```bash
curl -s https://owncast.online/install.sh | bash
```

**Nota**: No se sugiere que ejecutes esto como root, ya que estás descargando y ejecutando un script de internet. También se recomienda inspeccionar el contenido de cualquier script remoto antes de ejecutarlo.
