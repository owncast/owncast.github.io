Existen varias formas de instalar Owncast. El instalador rápido a continuación es el más fácil y funciona en la mayoría de los servidores Linux y macOS. También puedes instalar [manualmente](#manual-installation) desde una descarga de lanzamiento, ejecutar la [imagen del contenedor](/docs/getting-started/install/container), o usar una [instalación con un clic en un proveedor de alojamiento](/docs/getting-started/install/providers).
