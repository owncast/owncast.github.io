1. Asegúrate de tener [video passthrough](/docs/video/#video-passthrough) **desactivado**. Sé que quieres mantenerlo activado, pero si tienes problemas **DESACTÍVALO**. Si eso no lo soluciona, puedes volver a activarlo más tarde cuando resuelvas el problema.
2. Mira los registros de Owncast y el archivo `transcoder.log` para ver si hay errores que puedan ser útiles.
3. Para solucionar problemas, prueba con una fuente diferente y/o un reproductor diferente, para que cuando pidas ayuda sepas que funciona con X pero no con Y. Tu transmisión está disponible en `/hls/stream.m3u8`, así que intenta poner eso en VLC, Quicktime, etc.
4. Si estás utilizando un servicio de re transmisión de terceros o un servicio de transmisión basado en la web, intenta transmitir sin ella para ver si funciona. Una vez más, esto ayuda al pedir ayuda a hacernos saber dónde está el problema.
