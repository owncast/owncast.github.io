description: ''

El chat solo está habilitado cuando hay una transmisión activa. Esto es para detener mensajes de chat spam de personas cuando no hay transmisión.

Si estás usando un proxy frente a tu instancia de Owncast, asegúrate de que esté configurado correctamente para soportar websockets. Por defecto, algunos no pasan el websocket correctamente. Lee [la documentación de tu proxy](/docs/sslproxies/nginx) para asegurarte de que el soporte de websocket esté configurado correctamente para soportar el chat de Owncast.
