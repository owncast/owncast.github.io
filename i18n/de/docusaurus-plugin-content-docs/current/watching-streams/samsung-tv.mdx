      <li>Eingeschränkte Browserfähigkeiten auf den meisten Samsung-TVs.</li>

    </ul>

    <details>
      <summary>Schritt-für-Schritt-Anleitungen anzeigen</summary>

      1. Öffnen Sie die **Webbrowser**-App auf Ihrem Samsung TV (falls verfügbar).
      2. Navigieren Sie zu Ihrer Owncast-Server-URL.
      3. Versuchen Sie, den Stream direkt im Browser abzuspielen.

      **Hinweis:** Samsung-TVs haben eingeschränkte Unterstützung für webbasierte Video-Streaming und diese Methode funktioniert möglicherweise nicht zuverlässig.

    </details>

  </DeviceOptionCard>

  <DeviceOptionCard id="casting-samsung" title="Screen Mirroring/Casting" difficulty="mittel" reliability="mittel" screenshot="/images/devices/samsung-tv/screen-mirroring.png" screenshotAlt="Samsung TV Screen Mirroring Einrichtung" summary="Verwenden Sie Screen Mirroring von Ihrem Telefon oder Computer, um Owncast-Streams anzuzeigen.">








    <ul>
      <li>Funktioniert mit der Smart View-App von Samsung oder integriertem Screen Mirroring.</li>
      <li>Benötigt ein kompatibles Smartphone oder einen Computer.</li>
      <li>Kann eine gewisse Latenz einführen.</li>
    </ul>

    <details>
      <summary>Schritt-für-Schritt-Anleitungen anzeigen</summary>

      1. Stellen Sie sicher, dass Ihr Samsung TV und Gerät im selben WLAN sind.
      2. Öffnen Sie den Owncast-Stream auf Ihrem Telefon oder Computer.
      3. Verwenden Sie die Screen Mirroring-Funktionen:
         * **Android**: Verwenden Sie Smart View oder Screen Mirroring
         * **iOS**: Verwenden Sie AirPlay (wenn vom TV unterstützt)
         * **Windows**: Verwenden Sie Projektieren/Casten auf Gerät
      4. Wählen Sie Ihren Samsung TV aus der Liste der verfügbaren Geräte aus.

    </details>

  </DeviceOptionCard>

    * Apple TV
    * Roku
    * Amazon Fire TV Stick
    * Google Chromecast
    * NVIDIA Shield TV

    Befolgen Sie dann die Anweisungen für Ihr ausgewähltes Streaming-Gerät.

  </DeviceOptionCard>
