      <li>Funktioniert mit jeder Kodi-Installation.</li>
      <li>Direkter Zugang zu bestimmten Owncast-Servern.</li>
      <li>Keine zusätzlichen Plugins erforderlich.</li>
    </ul>

    <details>
      <summary>Schritte zur Anleitung anzeigen</summary>

      1. Gehen Sie in Kodi zu **Videos** > **Dateien**.
      2. Wählen Sie **Videos hinzufügen**.
      3. Wählen Sie **Durchsuchen** und dann **Netzwerkstandort hinzufügen**.
      4. Geben Sie die URL Ihres Owncast-Servers ein (z.B. `https://watch.owncast.online`).
      5. Speichern Sie den Standort und greifen Sie darauf zu, um den Stream abzuspielen.
