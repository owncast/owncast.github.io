  <DeviceOptionCard id="satv-webos" title="SATV-App" badge="Empfohlen" difficulty="mittel" reliability="mittel" screenshot="/images/devices/lg-webos/satv-app.png" screenshotAlt="SATV-App IPTV-Playlist-Schnittstelle" summary="Verwenden Sie die SATV-App, um das Owncast-Verzeichnis über die IPTV-Playlist auf Ihrem LG WebOS TV zu durchsuchen.">









    <ul>
      <li>Durchsuchen Sie das gesamte Owncast-Verzeichnis direkt auf Ihrem Fernseher.</li>
      <li>Kostenlose App im LG Content Store erhältlich.</li>
      <li>Automatische Aktualisierung der Live-Streams.</li>
    </ul>

    <details>
      <summary>Schritt-für-Schritt-Anleitungen anzeigen</summary>

      1. Installieren Sie **SATV** (kostenlos) aus dem App-Store Ihres LG-Fernsehers.
      2. Starten Sie es und drücken Sie die Taste auf **"Playlist hinzufügen"**.
      3. Geben Sie ein: `https://owncast.directory/api/iptv`. Stellen Sie sicher, dass es https ist.
      4. Überprüfen Sie, ob Sie es richtig eingegeben haben.
      5. Speichern Sie diese Playlist.
      6. Es aktualisiert die aktuellen Live-Streams jedes Mal, wenn Sie die SATV-App starten.
      7. Wählen Sie den Live-Stream aus, den Sie sehen möchten.

    </details>

  </DeviceOptionCard>

  <DeviceOptionCard id="direct-url-webos" title="Direkte URL-Eingabe" difficulty="schwierig" reliability="niedrig" screenshot="/images/devices/lg-webos/direct-url.png" screenshotAlt="WebOS-Browser direkte URL-Eingabe" summary="Einige WebOS-Browser unterstützen möglicherweise die direkte Wiedergabe von Owncast-Streams.">








    <ul>
      <li>Funktioniert möglicherweise mit dem integrierten Browser auf neueren LG TVs.</li>
      <li>Erfordert die manuelle URL-Eingabe.</li>
      <li>Begrenzte Kompatibilität und Zuverlässigkeit.</li>
    </ul>

    <details>
      <summary>Schritt-für-Schritt-Anleitungen anzeigen</summary>

      1. Öffnen Sie den Webbrowser auf Ihrem LG WebOS TV (falls verfügbar).
      2. Navigieren Sie zur URL Ihres Owncast-Servers.
      3. Versuchen Sie, den Stream direkt im Browser abzuspielen.

      **Hinweis:** Diese Methode hat begrenzten Erfolg und funktioniert möglicherweise nicht auf allen LG-TV-Modellen.

    </details>

  </DeviceOptionCard>
