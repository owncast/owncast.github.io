## Web Player

<img src="/docs/img/owncast-web-interface.png" alt="Die Owncast-Webschnittstelle zeigt den Video-Player, Streamtitel, Folgen- und Benachrichtigungsschaltflächen sowie den Live-Chat an" width="80%" />

Der Owncast-Webvideoplayer verfügt über eine Handvoll Tastenkombinationen, die Sie verwenden können.

| Aktion              | Tastenkombination |
| ------------------- | ----------------- |
| Abspielen/Pause     | *Leertaste*       |
| Lauter              | *0*               |
| Leiser              | *9*               |
| Stummschalten       | *m*               |
| Vollbild umschalten | *f*               |
| Chat umschalten     | *c*               |
