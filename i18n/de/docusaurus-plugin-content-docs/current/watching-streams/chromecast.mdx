    <ul>
      <li>Funktioniert mit jedem Owncast-Stream im Chrome-Browser.</li>
      <li>Einfacher Casting-Prozess mit nur wenigen Klicks.</li>
      <li>Keine zusätzlichen Apps auf dem Chromecast erforderlich.</li>
    </ul>

    <details>
      <summary>Schritt-für-Schritt-Anleitungen anzeigen</summary>

      1. Öffnen Sie **Chrome** auf Ihrem Computer.
      2. Besuchen Sie den Owncast-Server, den Sie ansehen möchten.
      3. Klicken Sie oben rechts auf die Schaltfläche **Mehr** und dann auf **Cast**.
      4. Wählen Sie das Chromecast-Gerät aus, auf dem Sie den Inhalt ansehen möchten.
      5. Wenn Sie das Chromecast bereits verwenden, wird Ihr Inhalt das, was auf Ihrem Fernseher ist, ersetzen.

    </details>

    Für weitere Informationen besuchen Sie die [offizielle Chromecast-Dokumentation](https://support.google.com/chromecast/answer/3228332).
