## Übersicht

Owncast umfasst eine Webschnittstelle für Ihr Video mit integriertem Chat, die verfügbar ist, sobald Sie den Server starten. Es zeigt Online-/Offline-Zustände, Zuschauerzahlen, Stream-Dauer, die Beschreibung Ihrer Instanz, Bilder, Links und mehr. Sie können es einfach ohne Änderungen verwenden, aber wahrscheinlich möchten Sie den Inhalt aktualisieren, der auf Ihrer Seite angezeigt wird, indem Sie Ihre Server-Admin-Seite besuchen.

<img src="/docs/img/owncast-web-interface.png" alt="The Owncast web interface showing the video player, stream title, follow and notify buttons, and live chat" width="80%" />

Zusätzlich wurde die Webschnittstelle speziell für Benutzer entwickelt, die mit der Anpassung von Farben und Stilen vertraut sind. Es wird keine Entwicklungsumgebung benötigt, öffnen Sie einfach die Verwaltung und beginnen Sie mit der Anpassung.

Wenn Sie Owncast in Ihre bestehende Webseite einbetten möchten, werfen Sie einen Blick auf unsere [Dokumentation zum Einbetten von Owncast](/docs/embed/).

Unten finden Sie einige Elemente, die Sie wahrscheinlich anpassen möchten, um den Inhalt zu aktualisieren, der auf Ihrer Seite angezeigt wird. Sie finden alle auf der **Allgemein**-Seite der Verwaltung.

<img src="/docs/img/admin-website-general.png" alt="The General settings page in the admin, with fields for the instance name, description, logo, tags, social links, and custom page content" width="65%" />

### Name und Beschreibung

Indem Sie Ihren Namen, Ihre Beschreibung und Ihr Logo festlegen, können Sie schnell den Inhalt der Website aktualisieren, um Ihren Stream widerzuspiegeln.

### Tags

Indem Sie Tags festlegen, zeigen Sie potenziellen Zuschauern, welche Kategorien von Inhalten Sie typischerweise streamen.

### Externe soziale Links

Sie können Links zu Ihren Profilen auf anderen Seiten hinzufügen, indem Sie sie im Admin hinzufügen.

### Inhaltsseite

Der Inhalt Ihrer Seite kann in Ihrem Admin-Bereich angepasst werden. Verwenden Sie die standardmäßige [Markdown-Syntax](https://www.markdownguide.org/basic-syntax/), um Links, Bilder und mehr hinzuzufügen.
