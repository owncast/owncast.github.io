Für die meisten Menschen wird Owncast direkt nach der Installation ohne weitere Konfiguration vollständig nutzbar sein. Wenn Sie einfach dem [Schnellstart](/quickstart) folgen, können Sie in wenigen Minuten streamen.

<img src="/docs/img/admin-dashboard.png" alt="Das Owncast-Admin-Dashboard, das die Navigationsseite und die Übersicht über den Streamstatus zeigt" width="80%" />

Das Admin-Panel ist der Ort, an dem Sie alles unten konfigurieren. Öffnen Sie es unter `/admin` auf Ihrem Server und melden Sie sich mit Ihrem Admin-Passwort an.

Es gibt jedoch eine Handvoll Elemente, die Sie konfigurieren können, um den Inhalt Ihrer Seite, [die Videoqualität](/docs/video), [Serverleistung](/docs/metrics) und mehr anzupassen.

Sie können auch die Funktionalität von Owncast erweitern, indem Sie Ihre eigenen Bots, Overlays, Tools und Integrationen erstellen und die APIs von Drittanbietern nutzen.
