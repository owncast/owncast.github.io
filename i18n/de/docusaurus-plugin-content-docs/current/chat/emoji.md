Der Chat Ihres Streams kann mit Ihren eigenen benutzerdefinierten Emojis angepasst werden. Sie können so viele hinzufügen, wie Sie möchten, und sie werden im Chat verfügbar sein.

## Im Admin-Bereich

Durch den Besuch der Seite für benutzerdefinierte Emojis im Admin-Bereich unter `/admin/chat/emojis` können Sie Bilder hinzufügen und entfernen, die für Chat-Teilnehmer verfügbar sind.

<img src="/docs/img/admin-custom-emoji.png" alt="Die Seite für benutzerdefinierte Emojis im Admin-Bereich, mit einer Schaltfläche zum Hochladen eines neuen Emojis über einem Raster der derzeit installierten Emojis" width="60%" />

## Manuelle Verwaltung

Sie können Ihre benutzerdefinierten Emoji-Bilder im Verzeichnis `data/emoji` auf Ihrem Dateisystem verwalten. Das können Sie tun:

- Mehrere statische und/oder animierte Emoji-Bilddateien auf einmal hinzufügen
- Ein ganzes Set von Emojis löschen
- Alle Emoji-Dateien umbenennen
- Sichern Sie Ihre benutzerdefinierten Emojis, indem Sie sie in einen anderen Ordner kopieren

## Bündelbilder von Emojis

Die mit Owncast ausgelieferten Bilder sind von verschiedenen Autoren frei lizenziert. Siehe das [Verzeichnis der Emojis](https://github.com/owncast/owncast/tree/develop/static/img/emoji) für die jeweiligen Lizenzen, die mit jeder Sammlung verbunden sind.
