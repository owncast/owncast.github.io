### Textformatierung

Der Web-Chat unterstützt einige grundlegende Formatierungen mit [Markdown](https://www.markdownguide.org/basic-syntax/):

Kursiv: `*Ihr Text*`

Fett: `**Ihr Text**`

Durchgestrichen: `~~Ihr Text~~`

Codeblöcke: <code>\`Ihr Text\`</code>

Wenn Sie eine Nachricht senden, rendert Owncast die Formatierung inline:
