Es gibt mehrere Möglichkeiten, Owncast zu installieren. Der schnelle Installer unten ist der einfachste und funktioniert auf den meisten Linux- und macOS-Servern. Sie können auch [manuell](#manual-installation) aus einem Release-Download installieren, das [Container-Image](/docs/getting-started/install/container) ausführen oder eine [One-Click-Installation bei einem Hosting-Anbieter](/docs/getting-started/install/providers) nutzen.
