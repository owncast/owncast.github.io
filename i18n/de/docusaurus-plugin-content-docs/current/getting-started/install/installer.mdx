Durch die Verwendung des Installers können Sie in etwa einer Minute startklar sein. Wenn Sie jedoch lieber [ein Release manuell herunterlädt](/docs/getting-started/install/manual), verwenden Sie eine [Ein-Klick-Installation](/docs/getting-started/install/providers) bei einem unterstützten Hosting-Anbieter oder andere Methoden, besuchen Sie die [Installationsseite](/docs/getting-started/install/), um die Installationsmethode zu finden, die am besten zu Ihren Bedürfnissen passt.

### Installer

Fügen Sie das Folgende in Ihr Terminal ein, und der Installer lädt die aktuellste Version von Owncast für Ihre Plattform herunter.
Es wird auch eine Kopie von ffmpeg heruntergeladen, falls Sie derzeit keine installiert haben.

```bash
curl -s https://owncast.online/install.sh | bash
```

**Hinweis**: Es wird nicht empfohlen, dies als root auszuführen, da Sie ein Skript aus dem Internet herunterladen und ausführen. Es wird auch empfohlen, den Inhalt jedes Remote-Skripts vor der Ausführung zu überprüfen.
