[Jitsi](https://jitsi.org) ist sowohl ein Anbieter für Video-Conferencing als auch eine Software-Suite. Es ist Open Source und kann selbst gehostet werden. Es ist auch als Dienst unter [meet.jit.si](https://meet.jit.si) verfügbar.

1. Besuche deine Jitsi-Meeting-Seite.
2. Klicke auf die drei Punkte in der unteren rechten Ecke.
3. Wähle "Live-Stream starten" aus.

![Image](/docs/img/jitsi-1.png)

4. Gib im Modal "Live-Stream starten" deinen vollständigen Owncast RTMP im Feld "Live-Stream-Schlüssel" ein. Das hätte das Format rtmp://dein-owncast-server/live/deinschlüssel.

![Image](/docs/img/jitsi-2.png)

5. Beachte, dass der Fehler "Live-Stream-Schlüssel könnte falsch sein" angezeigt wird. Das ist zu erwarten und kann ignoriert werden.
