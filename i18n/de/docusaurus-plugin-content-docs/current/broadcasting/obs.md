OBS ist ein beliebtes kostenloses Programm, mit dem Sie sofort von Ihrem eigenen Computer aus streamen können.

1. Installieren Sie [OBS](https://obsproject.com/) oder [Streamlabs OBS](https://streamlabs.com/) und bringen Sie es mit Ihrer lokalen Einrichtung zum Laufen.
2. Öffnen Sie die OBS-Einstellungen und gehen Sie zu "Stream".
3. Wählen Sie "Benutzerdefiniert..." als den Dienst.
4. Geben Sie die URL des Servers ein, auf dem Ihr Streaming-Dienst läuft, im Format rtmp://myserver.net/live.
5. Geben Sie Ihren "Stream Key" ein, der mit einem Stream-Schlüssel übereinstimmt, der im Owncast-Admin definiert ist. Standardmäßig gibt es einen einzigen Stream-Schlüssel, der mit dem Standard-Admin-Passwort `abc123` übereinstimmt, aber dies sollte geändert werden.
6. Starten Sie den Server.
7. Drücken Sie auf "Streaming starten" (OBS) oder "Live gehen" (Streamlabs) in OBS.
