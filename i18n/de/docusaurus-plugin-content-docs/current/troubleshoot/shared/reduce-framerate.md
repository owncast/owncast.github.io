Je weniger Videoframes verarbeitet werden müssen, desto weniger Hardware-Ressourcen werden benötigt, um sie zu verarbeiten. Sie können sowohl die Framerate dessen, was Sie an Owncast senden, als auch die Framerate, die an die Zuschauer gesendet wird, reduzieren.

**Nachteil**: Die Reduzierung der Framerate kann die Qualität Ihres Streams für einige Inhalte sichtbar verringern.
