Wenn das Hauptproblem darin besteht, dass Ihr Server nicht in der Lage ist, die Anzahl Ihrer Zuschauer zu bewältigen, können Sie von Drittanbietern für Objekt-Speicher profitieren, sodass Ihre Zuschauer das Video von dort herunterladen, anstatt von Ihrem Server. Das bedeutet, dass es egal ist, ob Sie 1 oder 1000 Zuschauer haben, der Video-Datenverkehr von Ihrem Server genau gleich sein wird. Bedenken Sie, dass jeder Zuschauer weiterhin direkt auf Ihren Server für den Chat zugreift.

Damit können Sie das Video auf Ihrem Owncast-Server generieren, es jedoch von einem Anbieter ausliefern, der unbegrenzte Bandbreite und Kapazität zu geringen Kosten hat.

Mit diesem Setup benötigen Sie keine extra CPU oder einen leistungsstärkeren Server, um mehr Zuschauer zu unterstützen, da sie Ihren Server technisch nicht berühren, sobald das Video beginnt.

[Erfahren Sie mehr über die Konfiguration von externem Speicher mit Owncast](/docs/storage).
