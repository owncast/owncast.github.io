Ihr Stream ist mindestens so gut, wie das, was Sie an Owncast senden. Stellen Sie sicher, dass Sie mit einer Qualität senden, mit der Sie zufrieden sind. Lesen Sie unsere [Vorschläge für Broadcast-Software](/docs/broadcasting/obs), in denen einige spezifische Punkte für Ihre Software aufgeführt sind.

Sie möchten möglicherweise auch die Menge an CPU erhöhen, die verwendet wird, um das Video auf Ihrem Owncast-Server zu verarbeiten. Wenn Sie niedriger gehen, wird die Qualität sichtbar abnehmen. Stellen Sie jedoch sicher, dass Ihr Server mit einer höheren CPU-Auslastung umgehen kann, sonst erleben Sie verschiedene Probleme.
