Das Hinzufügen zusätzlicher, niedrigerer Qualitäten in Ihrer Owncast-Video-Konfiguration ermöglicht es diesen Personen, kleinere Videos herunterzuladen, wodurch das Herunterladen schneller erfolgt und weniger Pufferung auftritt. Niedrigere Bitrate, Bildrate und Auflösung sind verfügbare Kombinationen von Einstellungen, die Sie nutzen können, um Ihren Zuschauern Optionen mit niedrigerer Qualität anzubieten.

**Nachteil**: Das Hinzufügen zusätzlicher Qualitäten führt dazu, dass mehr Hardware-Ressourcen auf Ihrem Server verwendet werden, also achten Sie auf die CPU-Auslastung, während Sie diese Änderungen vornehmen.
