1. Stellen Sie sicher, dass Sie [Video-Passthrough](/docs/video/#video-passthrough) **deaktiviert** haben. Ich weiß, dass Sie es eingeschaltet lassen möchten, aber wenn Sie Probleme haben, **SCHALTEN SIE ES AUS**. Wenn das nicht hilft, können Sie es später wieder einschalten, wenn Sie das Problem herausgefunden haben.
2. Sehen Sie sich auch Ihre Owncast-Protokolle sowie `transcoder.log` an, um festzustellen, ob es Fehler gibt, die hilfreich sein könnten.
3. Um Probleme zu beheben, versuchen Sie eine andere Quelle und/oder einen anderen Player, damit Sie, wenn Sie um Hilfe bitten, wissen, dass es bei X funktioniert, aber bei Y nicht. Ihr Stream ist unter `/hls/stream.m3u8` verfügbar, versuchen Sie also, das in VLC, Quicktime usw. einzufügen.
4. Wenn Sie einen Drittanbieter-Re-Streaming- oder webbasierten Streaming-Dienst verwenden, versuchen Sie, ohne ihn zu streamen, um zu sehen, ob es funktioniert. Auch das hilft, wenn Sie um Hilfe bitten, um uns mitzuteilen, wo das Problem liegt.
