title: Chat ist offline
description: ''
tags:

---

Der Chat ist nur aktiviert, wenn ein Stream aktiv ist. Dies soll verhindern, dass Leute während eines inaktiven Streams Spam-Nachrichten im Chat senden.

Wenn Sie einen Proxy vor Ihrer Owncast-Instanz verwenden, stellen Sie sicher, dass dieser ordnungsgemäß konfiguriert ist, um Websockets zu unterstützen. Standardmäßig passieren einige die Websocket-Verbindung nicht richtig. Lesen Sie [Ihre Proxy-Dokumentation](/docs/sslproxies/nginx), um sicherzustellen, dass die Websocket-Unterstützung richtig konfiguriert ist, um den Owncast-Chat zu unterstützen.
