There are several ways to install Owncast. The quick installer below is the easiest and works on most Linux and macOS servers. You can also install [manually](#manual-installation) from a release download, run the [container image](/docs/getting-started/install/container), or use a [one-click install at a hosting provider](/docs/getting-started/install/providers).
