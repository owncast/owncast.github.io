## Web Player

The Owncast web video player has a handful of keyboard shortcuts you can use.

| Action             | Shortcut   |
| ------------------ | ---------- |
| Play/Pause         | _Spacebar_ |
| Volume up          | _0_        |
| Volume down        | _9_        |
| Mute               | _m_        |
| Toggle full screen | _f_        |
| Toggle chat        | _c_        |
