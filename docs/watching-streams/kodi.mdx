      <li>Direct access to specific Owncast servers.</li>
      <li>No additional plugins required.</li>
    </ul>

    <details>
      <summary>Show step-by-step instructions</summary>

      1. In Kodi, go to **Videos** > **Files**.
      2. Select **Add Videos**.
      3. Choose **Browse** and then **Add Network Location**.
      4. Enter your Owncast server URL (e.g., `https://watch.owncast.online`).
      5. Save the location and access it to play the stream.
